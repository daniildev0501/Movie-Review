🎬 **Movie Review Website**

A simple HTML-only project created as a learning exercise. This repository contains a minimal movie review page focused purely on structure and content, without styling or frameworks.

📚 **Purpose:**

* Practice basic HTML skills
* Understand page structure and semantic elements
* Focus on content rather than design

✨ **Features:**

* Pure HTML (no CSS or JavaScript)
* Clean and minimal structure
* Easy to read and modify
* Great for beginners learning web development

📌 This project is intended as a study example, not a finished product.
