🎬 Movie Review: Stranger Things (FreeCodeCamp Lab)

This project is a simple HTML-based movie review page created as part of a FreeCodeCamp lab exercise. It focuses on structuring content using semantic HTML without any styling or frameworks.

📚 Purpose

* Practice semantic HTML structure
* Learn how to organize content using `<main>`, `<h1>`, `<ul>`, and `<p>`
* Work with images and accessibility attributes
* Build a simple static review page

✨ Features

* Movie review layout using pure HTML
* Image embedding with `alt` text for accessibility
* Structured cast list using unordered list
* Simple rating display
* Focus on readable and well-organized content

🛠️ Technologies

* HTML5 only (no CSS or JavaScript)

📖 About the Project

This page contains a short review of *Stranger Things* Season 1, including a summary of the plot, a rating, and a full cast list. It was built as a learning exercise to understand how to structure real-world content in HTML.

⚠️ Note

This is a beginner-level project made for educational purposes and does not include styling or advanced functionality.
